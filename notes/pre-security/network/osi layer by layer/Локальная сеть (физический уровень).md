**Локальная сеть (ЛВС — Local Area Network, LAN)** — это система соединённых между собой устройств (компьютеров, серверов, сетевых принтеров и других периферийных устройств), расположенных в пределах ограниченной территории: офиса, дома, учебного заведения, предприятия и т. п.. Она обеспечивает обмен данными, доступ к ресурсам и совместную работу пользователей. Локальная сеть по сути своей делиться на **топологии**.

**Топология сети** — это конфигурация структуры сети, которая определяет способ физического расположения устройств (узлов) и их соединения линиями связи (кабелями, радиосигналами и т. д.). Она описывает, как устройства связаны между собой, и влияет на передачу данных, производительность, надёжность и возможности расширения сети.

**Виды топлогий**:
1) **Шина (Bus).** Все устройства подключены к одному общему кабелю (шине), а терминаторы на концах нужны для предотвращения отражения сигнала. Данные передаются по этой линии, и все устройства могут их получать.
	```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌────────────-─┐
│      ПК1    │    │      ПК2    │    │      ПК3    │    │      ПК4     │
│   (Узел 1)  │    │   (Узел 2)  │    │   (Узел 3)  │    │   (Узел 4)   │
└─────┬───────┘    └─────┬───────┘    └─────┬───────┘    └─────-┬───────┘
         │                  │                  │                   │
         │                  │                  │                   │
         +──────────────────+──────────────────+──────────────────-+
         |                                                         |
         +=============== Общий коаксиальный кабель ===============+
         |                                                   |
      [T] (Терминатор)                                 (Терминатор) [T]
	```

2) **Звезда (Star).** Каждое устройство подключено отдельным кабелем к центральному узлу — коммутатору или концентратору (хабу).
	```
                                 [ Сервер ]
                                     |
                                     |
                           +---------+---------+
                           |                   |
                     +-----+-----+       +-----+-----+
                     |   HUB /   |       |  Принтер  |
                     |  Switch   |       |  (Сетевой)|
                     |    (Центр)|       +-----------+
                     +-----+-----+
                           |
            +--------------+--------------+
            |              |              |            
         +--+--+        +--+--+        +--+--+        
         |     |        |     |        |     |        
         | ПК1 |        | ПК2 |        | ПК3 |        
         +-----+        +-----+        +-----+       
	```

3) **Кольцо.** Устройства соединены последовательно в замкнутый круг. Данные передаются по кольцу в одном направлении.
	```
                          +-------------+
                         /               \
                        /                 \
                  +----+----+           +----+----+
                  |   ПК 1   |          |   ПК 2   |
                  | (Узел A) |          | (Узел B) |
                  +----+----+           +----+----+
                       |                      |
                       |                      |
                       |                      |
                       |                      |
                  +----+----+           +----+----+
                  |   ПК 4   |          |   ПК 3   |
                  | (Узел D) |          | (Узел C) |
                  +----+----+           +----+----+
                        \                 /
                         \               /
                          +-------------+
	```

4) **Древовидная.** Иерархическая структура, которая объединяет несколько топологий «звезда».
	```
	                                 [ Главный коммутатор ]
                                    |     (Корень сети)  |
                                    |       Switch 1     |
                                    +---------+----------+
                                              |
                    +-------------------------+---------------------+
                    |                         |                     |
               [ Switch A ]              [ Switch B ]           [ Switch C ]
              (Ветвь 2 этаж)            (Ветвь 1 этаж)         (Ветвь 3 этаж)
                    |                         |                     |
            +-------+-------+           +-----+-----+       +-------+-------+
            |       |       |           |     |     |       |       |       |
          [ПК1]  [ПК2]  [ПК3]         [ПК4] [ПК5] [ПК6]   [ПК7]  [Принтер] [ПК8]
	```

5) **Ячеистая (сетка, Mesh).** Каждый узел подключён к нескольким другим, что обеспечивает множество путей для передачи данных. 
	```
	                 +---------+---------+
                   /          |          \
                  /           |           \
            [ A ]             |             [ B ]
            /  \              |              /  \
           /    \             |             /    \
          /      \            |            /      \
        [ C ]----[ D ]------[ E ]------[ F ]----[ G ]
          \      /            |            \      /
           \    /             |             \    /
            \  /              |              \  /
            [ H ]             |             [ I ]
                  \           |           /
                   \          |          /
                    \         |         /
                     +-----[ J ]------+
	```
6) **Смешанная (гибридная).** Сочетает разные топологии, например, «звезда» и «кольцо».
	```
	                            [ Главный Core Switch ]
                                            |
                       +--------------------+--------------------+
                       |                                         |
               [Коммутатор этаж 1]                       [Коммутатор этаж 2]
                    (Звезда)                                 (Звезда)
                       |                                         |
                +------+------+                           +-----+-----+
                |      |      |                           |     |     |
               ПК1    ПК2    ПК3                         ПК4   ПК5   ПК6
                |                                          |
            [Принтер]                             [Кольцевой сегмент]
                                                         /    \
                                                  [SwA]---[SwB]---[SwC]
                                                   |       |       |
                                                  ПК7     ПК8     ПК9
                                                   \       |       /
                                                    \------+------/
                                                      (замкнуто)
	```
