Для установки прам по умолчанию используется команда `umask`. Cинтаксис команды:
```bash
umask [options] [mask]
```
Основные опции:
- `-p` : выдаст текущие права папки в цифровом виде 
- `-S` :  также выдаст права но уже в буквенном формате
Если использовать оции, и после команды прописать права(также как и в обычном `chmod` или же в расширенных правах), то эти права установятся по умолчанию для новых файлов и каталогов, путем маскировки стандартных прав. Он работате глобально фильтр для текущего сеанса пользователя
